# Demo Scenarios

## Scenario 1: The Struggling Student
1. User uploads "Lecture_Week3.pdf" (English).
2. Asks for a **Summary** in Korean.
3. Highlights "Neural Networks" term.
4. Clicks **Explain Term**.

## Scenario 2: Quick Voice Query
1. User drives to campus.
2. Taps **Mic**.
3. Says "Summarize the last chat".
4. App responds with text summary.
