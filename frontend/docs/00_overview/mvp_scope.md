# MVP Scope

## Core Features (Must Have)
1. **Translation**: Text-to-text translation (KR/EN/UZ).
2. **Summarization**: Summarize text inputs or uploaded files.
3. **Term Explanation**: Context-aware definitions of academic terms.
4. **Voice Interaction**: Voice-to-Text (STT) for queries.
5. **File Support**: PDF/TXT upload and processing.

## Out of Scope (For Now)
- Telegram Bot Integration (Postponed).
- Real-time video translation.
- Social features (User profiles, sharing).

## Technical Constraint
- No changes to existing UI design.
- Backend API must handle file parsing + AI logic.
