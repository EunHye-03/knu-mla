# Project Summary
## KNU Multilingual Assistant (KNU MLA)

### Vision
To provide KNU students with an intelligent assistant that breaks down language barriers in course materials, offering real-time translation, summarization, and term explanation.

### Core Value Proposition
- **Accessibility**: Understand complex lectures in your native language.
- **Efficiency**: Quickly summarize long documents and audio recordings.
- **Clarity**: Get instant explanations for difficult academic terms.

### Key Success Metrics
- Successful translation of PDF/TXT documents.
- accurate voice-to-text transcription.
- < 2s response time for text queries.
