# UI Components

## Atoms
- **Buttons**: Primary (Send), Secondary (Upload), Action (Translate/Summarize).
- **Inputs**: Text area, File input (hidden).
- **Icons**: Lucide-react (Mic, clip, send).

## Molecules
- **ChatBubble**: Displays user/AI messages with avatar.
- **InputArea**: Combined text input + action buttons.

## Organisms
- **Sidebar**: History list + Navigation.
- **ChatWindow**: Main conversation view.
